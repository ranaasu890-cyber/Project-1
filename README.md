# Project-1
sanke water Gun Game
1. Computer's choice

rand() % 3 gives a number: 0, 1, or 2

We map:

0 → snake

1 → water

2 → gun

2. Your input

You enter:

s → snake

w → water

g → gun

3. Game rules logic

This classic rule set:

Snake drinks water → Snake wins

Water ruins gun → Water wins

Gun kills snake → Gun wins

Same → Draw

4. Output

The program prints:

Computer’s choice

Whether you win/lose/draw
